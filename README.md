NAME
====

**Timezones::US** - Provides timezone data for the US for use with modules 'DateTime::US' and 'LocalTime::US'

SYNOPSIS
========

```raku
use Timezones::US;
```

DESCRIPTION
===========

**Timezones::US** exports constant US timezone data to be used by related date and time dynamic modules.
